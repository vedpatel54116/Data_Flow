Layer Responsibilities

Layer 0: Background
Animated mesh gradient. Glass samples this for color.
Layer 1: Glass Material
NSVisualEffectView with .sheet material. Provides blur + vibrancy.
Layer 2: Noise Texture
SVG fractalNoise at 3-5% opacity. Adds organic grain like the CSS sample.
Layer 3: Highlights
Top-edge gradient + inner box-shadow. Creates the "glass edge" depth.
